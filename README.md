# asan_demo
基于ASan（AddressSanitizer）的案例测试Demo，深度挖掘内存错误。

包含常用的内存错误测试案例，并配备详细的测试文档说明。


## 🛠️构建

```shell
mkdir build
cd build
cmake .. -DCMAKE_BUILD_TYPE=Debug
cmake --build .
```
