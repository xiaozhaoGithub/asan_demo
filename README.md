# asan_demo
基于AddressSanitizer（ASan）技术的案例测试Demo，深度挖掘内存错误检测能力。

本项目提供了一站式的测试环境，不仅包含了常用的ASan测试案例，还配备了详尽的测试文档说明，助力开发者轻松掌握ASan的应用技巧，提升软件质量。

适合对内存安全、错误检测及自动化测试感兴趣的开发者参考与学习。

## 🛠️构建

```shell
mkdir build
cd build
cmake .. -DCMAKE_BUILD_TYPE=Debug
cmake --build .
```
